## Steps

1. Clone the repository
2. Install the requirements

    ```sh
    poetry install
    ```

3. Run the `injest.ipynb` notebook
4. Run the streamlit app

    ```sh
    streamlit run app.py
    ```

5. Open your browser and navigate to [http://localhost:8501](http://localhost:8501)
